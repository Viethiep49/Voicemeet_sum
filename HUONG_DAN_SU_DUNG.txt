═══════════════════════════════════════════════════════════════
  BƯỚC 1: CÀI ĐẶT (CHỈ LÀM 1 LẦN ĐẦU TIÊN)
═══════════════════════════════════════════════════════════════

1. Cài đặt Python (nếu chưa có):
   - Truy cập: https://www.python.org/downloads/
   - Tải Python 3.11 hoặc 3.12
   - Khi cài đặt, NHỚ TICK vào "Add Python to PATH"
   - Cài đặt xong, khởi động lại máy tính

2. Chạy file SETUP.bat:
   - Double-click vào file: SETUP.bat
   - Đợi 10-15 phút (tự động cài đặt mọi thứ)
   - Xong!

═══════════════════════════════════════════════════════════════
  BƯỚC 2: CHẠY ỨNG DỤNG (MỖI KHI MUỐN DÙNG)
═══════════════════════════════════════════════════════════════

1. Double-click vào file: CHAY_APP.bat
2. Đợi 5-10 giây
3. Trình duyệt sẽ tự động mở
4. Upload file audio và bắt đầu!

═══════════════════════════════════════════════════════════════
  CÁC FILE QUAN TRỌNG
═══════════════════════════════════════════════════════════════

📁 SETUP.bat
   - Chạy 1 lần đầu tiên để cài đặt
   - Tự động cài đặt mọi thứ cần thiết

📁 CHAY_APP.bat  
   - Chạy mỗi khi muốn dùng ứng dụng
   - Tự động mở trình duyệt

📁 DEPLOYMENT\restart_server.bat
   - Dùng khi gặp lỗi hoặc server bị treo
   - Tự động restart server

═══════════════════════════════════════════════════════════════
  HƯỚNG DẪN SỬ DỤNG
═══════════════════════════════════════════════════════════════

1. Mở ứng dụng:
   - Double-click: CHAY_APP.bat
   - Đợi trình duyệt mở

2. Upload file audio:
   - Kéo thả file vào ô upload
   - Hoặc click để chọn file
   - Hỗ trợ: M4A, MP4, MP3, WAV, FLAC

3. Xử lý:
   - Click "Bắt đầu xử lý"
   - Đợi 5-15 phút (tùy độ dài file)
   - Xem progress bar

4. Tải kết quả:
   - Transcript: Nội dung đầy đủ
   - Summary: Tóm tắt

═══════════════════════════════════════════════════════════════
  XỬ LÝ LỖI
═══════════════════════════════════════════════════════════════

❌ Lỗi: "Python not found"
   → Cài đặt Python và tick "Add to PATH"
   → Khởi động lại máy tính

❌ Lỗi: "Port 8000 already in use"
   → Chạy: DEPLOYMENT\restart_server.bat

❌ Server bị treo hoặc không phản hồi
   → Nhấn Ctrl+C trong cửa sổ đang chạy
   → Chạy lại: CHAY_APP.bat

❌ Job bị kẹt ở một progress
   → Chạy: DEPLOYMENT\restart_server.bat
   → Upload lại file
