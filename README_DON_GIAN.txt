═══════════════════════════════════════════════════════════════
  BƯỚC 1: CÀI ĐẶT PYTHON (CHỈ LÀM 1 LẦN)
═══════════════════════════════════════════════════════════════

1. Truy cập: https://www.python.org/downloads/
2. Tải Python 3.11 hoặc 3.12 (64-bit)
3. Cài đặt và NHỚ TICK vào "Add Python to PATH"
4. Khởi động lại máy tính

═══════════════════════════════════════════════════════════════
  BƯỚC 2: CÀI ĐẶT ỨNG DỤNG (CHỈ LÀM 1 LẦN)
═══════════════════════════════════════════════════════════════

1. Double-click vào file: SETUP.bat
2. Đợi 10-15 phút (tự động cài đặt mọi thứ)
3. Xong!

═══════════════════════════════════════════════════════════════
  BƯỚC 3: CHẠY ỨNG DỤNG (MỖI KHI MUỐN DÙNG)
═══════════════════════════════════════════════════════════════

1. Double-click vào file: CHAY_APP.bat
2. Đợi 5-10 giây
3. Trình duyệt sẽ tự động mở
4. Upload file audio và bắt đầu!

═══════════════════════════════════════════════════════════════
  CÁC FILE QUAN TRỌNG
═══════════════════════════════════════════════════════════════

📁 SETUP.bat
   → Chạy 1 lần đầu tiên để cài đặt

📁 CHAY_APP.bat
   → Chạy mỗi khi muốn dùng ứng dụng

📁 DEPLOYMENT\restart_server.bat
   → Dùng khi gặp lỗi hoặc server bị treo

═══════════════════════════════════════════════════════════════
  LƯU Ý
═══════════════════════════════════════════════════════════════

- Lần đầu chạy SETUP.bat sẽ mất 10-15 phút
- Cần kết nối internet để tải models
- Model sẽ tự động tải khi upload file audio lần đầu
- Nếu gặp lỗi, chạy: DEPLOYMENT\restart_server.bat
